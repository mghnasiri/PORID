# PORID — OR Solver & Tool Navigator

The auto-updating guide to choosing OR optimization tools. Compare solvers, modeling languages, benchmarks, and licensing — all in one place.

**Live site:** [mghnasiri.github.io/PORID](https://mghnasiri.github.io/PORID/)

## What PORID Does

- **17 solvers compared** — Gurobi, CPLEX, HiGHS, SCIP, COPT, OR-Tools, and more
- **Decision helper** — answer your requirements, get a recommended solver stack
- **Compatibility matrix** — which modeling tools work with which solvers
- **Licensing guide** — academic license steps, pricing estimates, gotcha warnings
- **Performance data** — benchmark tiers sourced from Mittelmann benchmarks
- **Auto-updating** — daily pipeline pulls versions from GitHub Releases and PyPI

## Not a Paper Aggregator

PORID is focused on the tooling decision. It answers: "Which solver, modeling tool, and benchmark should I use for this optimization project?"

## Data Accuracy

Every data point is sourced. Editorial content (recommendations, learning curve ratings) is clearly labeled. See the [Methodology page](https://mghnasiri.github.io/PORID/methodology.html) for details.

Found an error? [Open an issue](https://github.com/mghnasiri/PORID/issues/new).

## Stack

- Frontend: Vanilla JS (ES6 modules), HTML5, CSS3
- Pipeline: Python 3.11, GitHub Actions (daily cron)
- Hosting: GitHub Pages
- Search: Fuse.js 7.0
